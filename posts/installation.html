Installation

Jott is installable from npm registry. It requires the global '-g' flag to run.

sudo npm install jott -g

To create a new Jott site, create a directory and execute the init command.

mkdir jott_site
cd jott_site
jott init

You will then be asked to an enter a name for your blog. This can be changed later by editing the jott.json file in your project's root directory.

Enter your Blog's name 
-> <Blog name>

Jott will then ask you for a baseUrl. Again, this can be changed in the jott.json file, and if you leave it blank, will default to http://localhost/project_dir

Enter your Blog's Base URL,
Eg: "localhost/blogtest" or "www.joeblogs.com"
(This can be changed in jott.json at anytime.)

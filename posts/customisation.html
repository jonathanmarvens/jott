Customisation

Jott comes with a readable, but very basic stylesheet. This can be found and customised here:

src/style/style.styl

All pages use a head and a footer, which are found in the templates file.

src/jade/templates/

The root page is index.jade

src/jade/index.jade

All pages are passed the jott.json file during jott build. Therefore any variable in jott.json is accessible using Jade's string interpolation. The only variables in there by default are 'title', and base url.

{
  "title" : "Jott",
  "baseUrl" : "http://jonsherrard.github.com/jott/",
  "googleAnalytics" : "UA-1234567-0",
  "variableName" : "Variable Value"
}

Any variable in the jott.json file, can be accessed as follows:


        h1 #{variableName}
        This will be rendered as:
        <h1>Variable Value</h1>
